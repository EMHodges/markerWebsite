Information Sheet Thank you for your interest in participating in this project. Please read the following information carefully. Purpose of the research The research project is to design an automated Python marker. The Python marker should work offline, be easy to use and able to mark composite questions (i.e. a question that consists of multiple parts, part I, part II etc.). Collecting data on how the marker awarded marks and whether the marker was usable will help me know if I’ve met my project’s aims. The study is being conducted as an undergraduate project for Computer Science Your participation You need to be over the age of 18. Participation is completely voluntary and you are free to withdraw your data at any point upto, and including, 10.04.22 without any negative consequences. Details on how to do so is in the ‘withdrawing consent’ section below. You will not be compensated for your participation, but your participation is very much appreciated. What will I need to do? You’ll need to download the python marker and run it locally on your machine. There will be a series of programming questions to attempt. Once finished you will be need to download a file containing your answers (the python marker will generate this file for you) and upload it to this website. There is also a questionnaire to fill in. What data will be collected? While answering questions you will be able to test your code and submit your answers as many times as you like. Each time you test your code or submit an answer your answer, marks awarded and generated feedback will be recorded. Note this data will only be used in the study and can only be seen by the researchers once the file generated by the automated python marker is uploaded to the website. You will be able to see the data collected by opening the file generated by the python marker. If you fill in the questionnaire your answers to these will also be collected. The questionnaire asks questions about the usability of the automated python marker and will not ask for any personal or identifying data. All the data will be anonymized. How will my data be anonymized? The name of the generated file containing your answers consist of a unique and randomized 6-digit number. All data supplied (confirmation of informed consent, file generated by the python marker and questionnaire answers) will be identified by this 6-digit number only. At no point will any personal or identifying information be asked for or recorded. Remember the name of this file since it will be needed to request for your data to be withdrawn as it is the only way to identify your submission. How will my data be stored? Your unique 6-digit id number, file generated by the automated marker and questionnaire answers will be stored in my university Google Drive. The University’s cloud storage solution is provided by Google, which means that data can be located at any of Google’s globally spread data centres. The University has data protection compliant arrangements in place with this provider (see https://www.york.ac.uk/it-services/google/policy/privacy/). The University processes personal data for research purposes under Article 6 (1) (e) of the General Data Protection Regulation (GDPR). The data will be kept until the final marks are formally approved for the module. If you have concerns about how your information is being processed, please contact the University’s Data Protection Officer at dataprotection@york.ac.uk. If you are concerned about the way in which the University has handled your personal data, you have a right to lodge a complaint to the Information Commissioner’s Office (Tel: 0303 123 1113, see www.ico.org.uk/concerns for more information). Who will see my answers? As the primary researchers of this project myself (Elizabeth Hodges) and my supervisor Lilian Blot will see your anonymized answers. Your anonymous answers may also be shared with other researchers at the University of York. If the data is published it will be completely anonymous and unidentifiable. How can I be sure downloading the marker is safe? The automated python marker is an exe file. The exe file is only available to download on this website once it has been scanned by Microsoft Defender with no threats being found. Withdrawing consent You are free to withdraw your data at any point and for any reason up to, and including, 10.04.22 by emailing emh566@york.ac.uk with your unique 6-digit ID (this will be included in the name of the generated download file submitted). Questions? Please email any questions to me: emh566@york.ac.uk or my supervisor Lilian Blot: lilian.blot@york.ac.uk. Consent
