 *Using Chrome is recommended
  1. Download the Python Marker
    • Click the Download button - do not open the downloaded file
    • You should see the downloaded file in the top right
      The warning message is because the file is not known to Microsoft edge - the python marker is not malicious and has been scanned with Microsoft Defender showing no threats
    • Hover over the download until an ellipsis appears
    • Click on the ellipsis and click 'Keep' from the dropdown menu
    • This will open a dialog box, click the chevron next to 'Show more' and then click 'Keep anyway'
    • Now you should be able to see the file without a warning. Click on the file icon to open File Explorer
  • Open powershell
    • In File Explorer highlight the text in the address bar (click on the white space to the left of the chevron in the address bar) - the text should turn blue as in the image below:

      • .\{{downloadId}}.exe runserver --noreload

    • Running the python marker on Mac

    on Windows?
    1. Download the Python Marker for Mac - see above for Windows instructions
      • Click the Download button for your Mac (Macs with the M1 chips are ARM64, others are x86 - if in doubt choose x86)
    2. Setup the Python Marker
      • Open terminal
        • One way to do this is to:
        • Open launchpad and search for terminal
        • Click on the terminal icon (it looks like the icon below) to open the terminal
          • The terminal window should look similar to the image below:
      • Once in the terminal type (or copy and paste) the commands below - press Enter between each command:
        • cd ~/Downloads
        • chmod 777 {{downloadId}}
        • xattr -dr com.apple.quarantine "{{downloadId}}"

        Your terminal window should look like the image below, though with {{downloadId}} instead of python-marker123456:

      For interest - the second command gives you the permission to run the python marker, the third command lets you run the python marker, without this Apple will complain that the file has come from an unidentified developer and won't let you run it
    3. Run the python marker
      • Type (or copy and paste) the command below and then press Enter:
        • ./{{downloadId}} runserver --noreload

        Your terminal window should look like the image below though with .\{{downloadId}} instead of .\python-marker123456
      • Wait until you see some text that looks similar to that in the image below:
      • Click the run button below to open the python marker (or navigate to 'http:127.0.0.1:8000')
